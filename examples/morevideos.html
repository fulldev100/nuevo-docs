More Videos Plugin

The plugin allows to overlay a gallery of suggested videos at the bottom of the player, above the control bar. It's almost same like a video galleryy on YouTube embedded player.
Before you initialize morevideos plugin you must define an array of suggested videos. You can do it in exactly same way as for related videos.

Initialize player and plugins


<script>
	var player = videojs('player_1');
	player.nuevo({ 
		option1: value,
		option2: value
	});
	var more_videos = [....];
	player.morevideo(more_videos); 
</script>
You can use different arrays or the same array of videos for related dialog and more videos gallery. You can disable related videos menu item and use only more videos gallery.
